# i Tunes Sales Analysis Dashboard

Project Overview
This project is an interactive Power BI dashboard built using the iTunes dataset.  
The dashboard analyzes music sales performance, customer activity, genre popularity, artist revenue contribution, and track-level insights.

The goal of this project is to transform raw transactional data into meaningful business insights using SQL and Power BI.

---

 Tools & Technologies Used

- MySQL
- Power BI
- SQL
- Data Modeling
- DAX Measures
- Data Visualization

#  Key Insights

## Revenue Insights
- Total revenue generated was approximately **1.63M**
- Revenue remained stable across most months
- Revenue dipped during October–November and recovered toward December

---

 Customer Insights
- Approximately **20K customers** contributed to platform sales
- Average invoice value was around **2.66K**

---

 Genre Insights
- Rock generated the highest revenue
- Metal and Blues were also strong-performing genres
- Smaller genres contributed relatively lower revenue

---

Artist Insights
Top-performing artists included:
- AC/DC
- Aerosmith
- Alanis Morissette

These artists generated significant portions of overall revenue.

---

 Track Insights
A small group of tracks contributed disproportionately to total revenue, indicating:
- strong customer preference concentration
- importance of playlist/recommendation systems

---

 Data Modeling

Relationships were created between:
- Invoice ↔ Customer
- Invoice Line ↔ Track
- Track ↔ Genre
- Track ↔ Album
- Album ↔ Artist

A star-schema-like structure was used for analysis.

---

# Business Recommendations

- Focus marketing campaigns on high-performing genres
- Promote top artists through playlists and recommendations
- Investigate seasonal revenue drops
- Improve visibility for low-performing genres
